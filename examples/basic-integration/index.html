Tessaliq Integration Example

Zero-knowledge age verification — the date of birth never leaves your browser.

Verify your age

How it works:
  1. The SDK creates a verification session with the Tessaliq API
  2. Your date of birth is used to generate a ZK proof locally in your browser
  3. Only the proof (not your DOB) is sent to the server for verification
  4. The server confirms: "this person is 18+" — nothing more
